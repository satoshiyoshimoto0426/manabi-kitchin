## 二次利用について

- 上記すべての素材は、学び舎キッチン内および協力団体との共有目的で自由に使えます。
- 動画にクレジットを入れる場合：「企画：学び舎キッチン / 制作：ManabiOps / 2026年4月」
- 画像・音声の再編集も可能です（assets/ にオリジナルファイル収録）。
- 外部公開（YouTube、SNS広告等）する場合は事前にOwnerに確認してください。

## 再生成・再編集のコマンド

```bash
# 紹介動画を再ビルド
python3 scripts/build_video.py

# 機能説明書/セットアップガイドをPDF化
python3 scripts/build_pdfs.py
```
