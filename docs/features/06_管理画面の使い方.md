# 🖥 管理画面の使い方

> **読了目安: 7分** / **想定読者: 運営代表・チェック担当**

---

## 📝 一言でいうと

> **ブラウザで ManabiOps の全体状況を確認・操作できる画面**
> LINE では難しい「細かい編集」「一覧表示」「データ削除」ができます。

---

## 🔗 アクセス方法

### URL

```
https://manabi-ops-xxxxxx-an.a.run.app/admin
```

※実際のURLは運営代表者にお問い合わせください。

### ログイン

ブラウザで開くと **ID・パスワード入力** を求められます (Basic認証)。

```
┌───────────────────────┐
│ 🔐 認証が必要です         │
│ ─────────────────── │
│  ユーザー名: [         ]  │
│  パスワード: [         ]  │
│                        │
│         [ログイン]        │
└───────────────────────┘
```

- ID/パスワードは運営代表者から共有されます
- **パスワードは絶対に他人に教えない**
- 必要なら2段階認証 (Cloudflare Access) を追加可

## 🧾 取引 (会計) 画面

### できること

- 全取引の一覧表示・検索
- 日付・科目・金額での絞り込み
- 個別取引の編集・削除
- CSV エクスポート
- 手入力フォーム

### 一覧画面

```

│ 🧾 取引一覧                                  │
│ ────────────────────────────────────── │
│ 検索: [             ] 期間: [2026/04 ▼]    │
```

### 個別編集

取引の **[編]** ボタンをクリック → 以下を変更可能:

- 金額
- 勘定科目
- 店舗名
- 品目詳細
- 日付
- 備考

変更すると Google Sheets 側も自動更新されます。

## 👥 参加者画面

### できること

- 参加者の一覧表示 (氏名は復号表示 — Owner のみ)
- 参加履歴・回数の確認
- 個別参加者の削除 (個人情報削除依頼対応)
- CSV エクスポート (氏名を含まない形式)

### 一覧画面

```

│ 検索: [氏名     ]  区分: [全て ▼]       │

│ ─────────────────────────────── │

│ p_002 田中花子  子ども 5回  2026/04/16 │
│ p_003 佐藤美咲  子ども 3回  2026/04/16 │
│  ...                                   │
└────────────────────────────────────┘
```

> 🔒 **氏名は Owner のみ表示**。Staff/Viewer には `●●●` とマスクされます。

### 個別削除 (削除権対応)

保護者から「削除してほしい」と連絡があった場合:

1. 氏名検索で該当者を特定
2. **[削除]** ボタンをクリック
3. 確認ダイアログ → **[完全削除]**
4. 削除理由を記録 → 実行

**物理削除で復元不可** です (法令対応)。

## 👤 ユーザー管理画面

### できること

- LINE ホワイトリストの追加・削除
- ロール (役割) 変更
- LINE ユーザーIDの確認

### 一覧画面

```
┌──────────────────────────────────┐
│ 👤 ユーザー管理                          │
│ ──────────────────────────────── │
│ + [新規追加]                          │
│                                      │
│ 表示名     ロール   LINE User ID       │
│ ─────────────────────────────── │
│ 田中代表   Owner    U_a1b2c3... [編]  │
│ 佐藤さん   Staff    U_d4e5f6... [編]  │
│ 山田さん   Viewer   U_g7h8i9... [削]  │
└──────────────────────────────────┘
```

### 新規追加

- 表示名 (分かりやすい名前)
- LINE User ID (スタッフから聞く)
- ロール (Owner / Staff / Viewer)

### ロール説明

| ロール | できること |
|--------|----------|
| **Owner** | すべての操作 (承認・削除・ユーザー管理) |
| **Staff** | 写真送信・編集・承認依頼 (削除は不可) |
| **Viewer** | 閲覧のみ (編集・承認不可) |

## 🆘 困ったとき

### Q. ログインできない

- ID・パスワードを再確認
- 運営代表者に確認 (Secret Manager の `admin-basic-auth`)
- ブラウザのキャッシュクリア

### Q. 画面が真っ白

- ブラウザを Chrome 最新版に更新
- スマホの場合、PCで再アクセス
- それでもダメならCloud Runログ確認 ([13_troubleshooting.md](../setup-guide/13_troubleshooting.md))

### Q. 取引を削除したら Google Sheets もおかしくなった

- Google Sheets の「版の管理」で過去バージョンに戻す
- 管理画面と Sheets の整合性を取るため、Sheets 側の直接編集は避ける

### Q. 氏名が `●●●` と表示される

- Owner ロール以外では氏名は見えない設計です
- Owner でもマスクされる場合 → セッション切れ。再ログイン

## 💡 よく使う操作

### 📌 月末の確認 (5分)

1. **ダッシュボード** で今月の合計確認
2. **承認待ち** に残がないか確認
3. **取引** から CSV エクスポート → 保存
4. **統計** で推移グラフ確認

### 📌 助成金申請用資料作成

1. **取引** で期間を指定 → CSV エクスポート
2. **統計** で参加者数・回数を確認
3. **SNS投稿** のエンゲージから活動実績を転記

### 📌 問題取引の調査

1. **取引** で検索・絞り込み
2. **[編]** ボタンから詳細確認
3. 必要なら元レシート画像を確認 (`レシートURL` 列)
