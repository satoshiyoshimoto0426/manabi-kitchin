## 🙏 このツールを作った理由

まなびキッチンは、毎月1〜4回の開催。
そのたびに運営代表者は **2〜3時間の事務作業** に追われていました。

- レシート整理・家計簿入力: 1時間
- 参加者名簿の Excel 入力: 30分
- 写真の顔ぼかし・SNS投稿: 1時間
- 月次報告書作成: 2時間

これらを **「LINEで写真を送るだけ」** に置き換えるのが ManabiOps の目的です。

> **削減できた時間は、子どもたちと向き合う時間に使いましょう。**
