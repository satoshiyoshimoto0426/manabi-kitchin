## 🪄 AI が自動でやってくれること

### 1. レシート自動読取 (OCR)

- Google の最新OCR技術 **Document AI** が使われます
- 日本のレシート・領収書に特化したモデルなので高精度
- 手書きの領収書もある程度読めます

### 2. 勘定科目の自動推測

「何費で登録するか」を AI が推測します。

| 品目例 | 推測される科目 |
|-------|-------------|
| 米・野菜・肉・調味料 | 食材費 |
| ラップ・洗剤・ふきん | 消耗品費 |
| ガソリン代 | 交通費 |
| 会場レンタル料 | 会場費 |
| おもちゃ・絵本 | 教育費 |
| その他 | 雑費 |

**間違った推測** がされたら、**[✏ 修正]** ボタンで正しい科目に変更できます。

### 3. 重複登録チェック

同じレシートを2回送ってしまった場合、ManabiOps が検知して警告します。

```
⚠️ 同じ日付・店舗・金額の記録が既にあります
   2026/04/16 ○○スーパー ¥4,820

   本当に登録しますか?
   [はい、別の取引] [いいえ、やめる]
```

## 🎯 何がすごいの? (従来との比較)

### 📊 従来の経理作業

```
 ① レシートを財布や封筒に貯める        (毎日)
 ② 月末、1枚ずつ取り出して並べる         (30分)
 ③ Excel を開く                         (1分)
 ④ 1枚ずつ入力                         (1時間)
 ⑤ 科目を考える・過去の記録を見返す      (30分)

 ⑦ 月次報告書に転記                    (30分)
 ─────────────────────
 合計: 約3時間/月
```

### 🚀 ManabiOps の経理作業

```

 ② 確認カードに [✅ 登録]                (5秒)
 ③ 月次: LINEで来たサマリを見る         (1分)
 ─────────────────────
 合計: 約3分/月 + 1レシートあたり15秒
```

## 🔐 会計データの信頼性

「AI が勝手に登録したら間違いがあるのでは?」という心配への答え:

### ✅ 必ず人間が承認する

- AI はあくまで「**下書き**」を作るだけ
- **[✅ 登録]** を押さない限り、Google Sheets には1行も追加されません
- 承認したのが誰なのかも記録されます (監査可能)

### ✅ いつでも修正できる

- Google Sheets 側で直接修正可能 (編集者権限)
- 管理画面 `/admin/transactions` からも修正可能
- 修正履歴は Google Sheets の「版の管理」で見られる

### ✅ 元レシート画像も保管

- 承認した取引には元画像のURLが記録されます
- 税務・助成金監査で「証拠を見せて」と言われても対応可

## 📋 記録されるフィールド一覧

Google Sheets に記録される10項目:

| 列 | 項目 | 例 | 用途 |
|:--:|----|-----|------|
| A | 日付 | 2026/04/16 | いつの取引か |
| B | 種別 | 支出/収入 | 収支の区別 |
| C | 勘定科目 | 食材費 | 費目分類 |
| D | 金額 | ¥4,820 | 税込合計 |
| E | 店舗 | ○○スーパー | どこで |
| F | 品目 | 米5kg, 人参3本… | 何を |
| G | 取引ID | tx_ac156a… | 内部管理用 |
| H | 承認者 | 田中さん | 誰が承認 |
| I | 承認日時 | 2026/04/16 18:32 | いつ承認 |
| J | レシートURL | (画像リンク) | 元画像 |

## 💡 応用テクニック

### 📌 寄付金を登録する場合

寄付金は「レシート」がないので、テキストで送ります。

```
👤 「寄付金 10,000円 匿名希望 2026/04/08」
🤖 「以下の内容で登録しますか?
      📅 2026/04/08
      💴 ¥10,000 (収入)
      📂 科目: 寄付金
      📝 備考: 匿名希望
     [✅ 登録] [✏ 修正] [❌ 取消]」
```

### 📌 複数店舗のレシートを連続で送る

3枚のレシートを連続で送ると、3つの確認カードが順番に来ます。
それぞれで **[✅ 登録]** をタップするだけ。

### 📌 月末の集計

LINE で以下のように打つと、簡易サマリが返ってきます。

```
👤 「サマリ」または「今月の集計」

🤖 ┌───────────────────┐
    │ 📊 2026年4月 (途中経過) │
    │                       │
    │ 💰 収入: ¥45,000       │
    │ 💸 支出: ¥32,150       │
    │ 📈 残高: +¥12,850     │
    │                       │
    │ 🧾 取引件数: 23件      │
    │ 👥 参加者累計: 87名   │
    └───────────────────┘
```

## 🆘 困ったとき

### Q. 金額が間違って読み取られた

- **[✏ 修正]** → 正しい金額を LINE で返信
- 例: `¥4,820` → 正しくは `¥4,280` なら、`4280` と送信

### Q. 品目を手入力で変更したい

- 管理画面 `/admin/transactions` から編集可能
- Google Sheets 側で直接書き換えてもOK

### Q. 現金をおろした・両替した場合

- 「現金振替」として手動登録 (管理画面の手入力フォーム)
- Sheets の「種別」列で「振替」と入力

### Q. ずっと前のレシートを登録したい

- LINE で日付をつけて送る: `2026/03/15 のレシートです` と写真と一緒に送信
- または管理画面の手入力フォームから登録
