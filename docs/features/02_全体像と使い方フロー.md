# 🗺️ 全体像と使い方フロー

> **読了目安: 5分** / **想定読者: 全員**

---

## 🏞 全体像

ManabiOps は、まなびキッチンの運営を **3つの時期** に分けて支えます。

```
    ┌──────────────┐   ┌──────────────┐   ┌──────────────┐
    │ ①イベント準備 │→│ ②イベント当日 │→│ ③イベント後   │
    └──────────────┘   └──────────────┘   └──────────────┘
         🛒                  📸                 📊
    買い出し・仕込み       写真撮影・受付       報告・記録
         ↓                    ↓                   ↓
    レシートを送る        名簿・写真を送る      自動で完了
         ↓                    ↓                   ↓
    [家計簿 自動記帳]    [参加者DB 登録]   [SNS投稿・月次レポート]
```

---

## 📱 基本の使い方: たった3ステップ

### Step 1. LINE で写真を送る

まなびキッチン公式アカウントに、**普通の友だちに送るのと同じように** 写真を送ります。

- レシート? → 自動で会計処理
- 名簿?   → 自動でデジタル化
- 写真?   → 自動で顔ぼかし + SNS案作成

**AIが勝手に内容を判別** するので、スタッフは **何も指定しなくてOK**。

### Step 2. 自動返信を確認

数秒〜30秒で、確認カードが返ってきます。

```
┌─────────────────────┐
│ 🧾 レシート確認          │
│ ─────────────────── │
│  ○○スーパー             │
│  2026/04/16             │
│  ¥4,820                 │
│                          │
│  勘定科目: 食材費         │
│  品目: 米5kg, 人参3本...  │
│                          │
│ [✅ 登録] [✏ 修正] [❌ 取消] │
└─────────────────────┘
```

### Step 3. ボタンをタップ

内容が合っていれば **[✅ 登録]** をタップ。間違っていれば **[✏ 修正]**。

→ 以降は **自動** で Google Sheets や参加者DB、Instagram に反映されます。

## 👥 役割別の使い方

### 👑 運営代表者 (Owner)

**やること**: 承認・全体確認・月次レポート受取

| 場面 | やること | 時間 |
|------|---------|:---:|
| レシート来た | [✅ 登録] をタップ | 10秒 |
| 名簿来た | [✅ 登録] をタップ | 10秒 |
| 写真来た | キャプション確認 → [✅ 投稿] | 30秒 |
| 月初 1日 | LINE で月次サマリを受信・確認 | 2分 |
| 異変時 | 管理画面で詳細確認 | 5分 |

**1ヶ月あたりの操作時間: 約30分**

---

### 🍳 スタッフ (Staff)

**やること**: 写真を送る (承認はOwnerに任せる)

| 場面 | やること | 時間 |
|------|---------|:---:|
| 買い物後 | レシート写真を送る | 10秒 |
| 受付時 | 名簿を撮って送る | 10秒 |
| イベント中 | 写真を気軽に送る | 5秒/枚 |

**1イベントあたりの操作時間: 約3分**

---

### 🙋 ボランティア (Viewer)

**やること**: 気軽に写真を送る

承認権限は無いので、送った写真が勝手に投稿されることはありません。
Owner が確認してから投稿されます。

## 🔄 裏側で起きていること (しくみ)

気になる方向けの、**自動処理の流れ**。

```
 [LINE]
   │ 写真送信
   ▼
 [ManabiOps 受信]
   │
   ├─ これは何の写真? ─────→ [Gemini AI が判別]
   │                               │
   │                               ├─ レシート → [Document AI でOCR]
   │                               │              ↓ 金額・店名・品目を抽出
   │                               │            [勘定科目を AI が推測]
   │                               │              ↓
   │                               │            [Flex Message で確認依頼]
   │                               │              ↓ 承認
   │                               │            [Google Sheets に記帳]
   │                               │
   │                               ├─ 名簿 → [Document AI で OCR]
   │                               │           ↓ 氏名・人数を抽出
   │                               │         [暗号化して Firestore 保存]
   │                               │
   │                               └─ 写真 → [顔ぼかし処理]
   │                                         ↓
   │                                       [キャプション AI 生成]
   │                                         ↓ 承認
   │                                       [Instagram に投稿]
   │
   └─ 結果を LINE に返信
```

スタッフは **この流れを意識する必要はありません**。
「LINEで送る」「タップする」だけ。

## 📅 1ヶ月の使い方タイムライン

```
  月初1日 9:00    LINE: 「前月サマリが届きました」(Owner宛)
                     ↓
   (中旬 イベント準備期間)
                     ↓
  ●買い出しの日      LINE: レシート送る → 即 家計簿に記帳
                     ↓
  ●イベント開催日    LINE: 名簿・写真送る → 自動処理
                     ↓
  (イベント後)       LINE: 承認カードが来る → タップ
                     ↓
  (翌朝)            Instagram に投稿が反映
                     ↓
  翌月1日 9:00      LINE: 月次サマリが届く
```

## 🏁 成果物: どこで結果を見られる?

| 見たいもの | 場所 | 誰が見れる |
|----------|------|----------|
| 月次サマリ | LINE (毎月1日) | Owner |
| 会計台帳 | Google Sheets | Owner + 指名スタッフ |
| 参加者リスト | 管理画面 `/admin` | Owner のみ (暗号化済) |
| 過去のSNS投稿 | Instagram アプリ | 公開・だれでも |
| グラフ・集計 | Looker Studio | Owner + 指名スタッフ |
| 承認待ち一覧 | 管理画面 `/admin` | Owner |
