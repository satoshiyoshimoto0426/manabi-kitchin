# 📘 これは何? — ManabiOps を1枚で

> **読了目安: 3分** / **想定読者: 全員 (初めて聞く方向け)**

---

## 💡 一言でいうと

> **LINEで写真を送るだけで、こども食堂の事務作業がぜんぶ自動化されるシステム**

それが「**ManabiOps** (まなびオプス)」です。

---

## 🎯 なぜ作ったの?

まなびキッチンの運営代表者は、毎月こんな作業に追われていました。

### 😫 Before (ManabiOps 導入前)

```
  イベント当日                  翌日〜月末
 ┌─────────┐              ┌──────────────────┐
 │  楽しい時間 │              │ 🧾 レシート整理   (1時間)│
 │  子どもたちと│              │ 💻 Excelに入力    (30分) │
 │  保護者対応 │ ──→ 終了 ──→│ 📸 写真選別・顔加工 (1時間) │
 │  料理・片付け│              │ 📱 Instagram投稿  (30分) │
 └─────────┘              │ 📊 月次報告書作成 (2時間) │
                              │ ─────────────── │
                              │  合計: 月5〜8時間  │
                              └──────────────────┘
```

**運営代表者がパソコンに向かう時間**が、子どもと向き合う時間を圧迫していたのです。

### 😊 After (ManabiOps 導入後)

```
  イベント当日                  翌日〜月末
 ┌─────────┐              ┌──────────────────┐
 │  楽しい時間 │              │ 📱 LINE で写真送る (10秒) │
 │  子どもたちと│              │ 🤖 AIが自動処理          │
 │  保護者対応 │ ──→ 終了 ──→│ 👆 LINEで [✅登録] タップ │
 │  料理・片付け│              │ ─────────────── │
 │  📱写真送る │              │  合計: 月 約30分          │
 └─────────┘              └──────────────────┘
                                     ↓ 削減
                              🎉 月 約 5〜7時間 の余裕
```

削減できた時間で **子どもたちと向き合えるように** なります。

---

## 🪄 具体的に何をしてくれるの?

LINE で **「まなびキッチン」公式アカウント** に写真を送るだけで、AI が中身を判別して、すべてを自動化します。

### 1️⃣ レシートを送ると…

```
👤 スタッフ:       📱 [レシート写真を送信]
                          ↓
🤖 ManabiOps:     「○○スーパー ¥4,820
                     食材費として登録しますか?
                     米5kg, 人参3本, 玉ねぎ...」
                     [✅ 登録] [✏ 修正] [❌ 取消]
                          ↓ [✅ 登録] タップ
📊 Google Sheets: 自動的に家計簿に1行追加される
```

### 2️⃣ 名簿 (手書きOK) を送ると…

```
👤 スタッフ:       📱 [手書き名簿の写真を送信]
                          ↓
🤖 ManabiOps:     「参加者12名を読み取りました:
                     田中太郎(大人), 田中花子(子ども), ...
                     登録しますか?」
                     [✅ 登録] [✏ 修正]
                          ↓ [✅ 登録] タップ
📂 参加者DB:       自動的に暗号化して保存される
                   (氏名は絶対に他人から見えない)
```

### 3️⃣ イベント写真を送ると…

```
👤 スタッフ:       📱 [子どもたちが写った写真を送信]
                          ↓
🤖 ManabiOps:     「顔を自動でぼかしました ✨
                     キャプション候補を作りました:
                     『今日も笑顔いっぱいのまなびキッチン!
                      #こども食堂 #手作りごはん』」
                     [✅ Instagram投稿] [✏ 修正] [❌ 取消]
                          ↓ [✅ 投稿] タップ
📱 Instagram:     30秒以内に投稿完了
```

## ⏱ どのくらい楽になるの?

| 作業 | Before | After | 削減 |
|------|:------:|:-----:|:----:|
| レシート整理・入力 | 1時間 | 1分×枚数 | **−95%** |
| 名簿入力 | 30分 | 3分 | **−90%** |
| 写真の顔ぼかし | 1時間 | 0分 (自動) | **−100%** |
| SNS 投稿 | 30分 | 30秒 | **−99%** |
| 月次報告書 | 2時間 | 自動生成 | **−100%** |
| **合計/月** | **5〜8時間** | **30分** | **約 −90%** |

---

## 🛡 安全・安心への配慮

- **顔は 100% ぼかされて** から投稿候補になる (子ども・保護者の顔は絶対に流出しない設計)
- **名前は暗号化** されて保存 (仮にデータが漏れても読めない)
- **投稿前に必ず人間が承認** する (AIが勝手に投稿することはない)
- **写真は1年後に自動削除** される (いつまでも残らない)

→ 詳しくは [08_安心・安全への配慮.md](./08_安心・安全への配慮.md)

## 👥 誰が使うの?

| 役割 | やること |
|------|---------|
| **運営代表 (Owner)** | すべての機能を使う / 承認権限あり |
| **スタッフ (Staff)** | LINE で写真を送る / 承認は Owner に任せる |
| **ボランティア (Viewer)** | LINE で写真を送るだけ |
| **保護者・参加者** | 使わない (情報が登録されるだけ) |

→ 詳しい権限は [09_費用と運用体制.md](./09_費用と運用体制.md#権限)

## 📖 用語集

このガイドに出てくる IT 用語のやさしい解説です。

| 用語 | やさしく言うと |
|------|-----------|
| **AI (人工知能)** | コンピュータが、写真の中身を見て「これはレシートだな」と判別する技術 |
| **LINE公式アカウント** | お店や団体が持てる LINE。普通の友だちLINEと同じように使える |
| **Webhook (ウェブフック)** | LINE に来たメッセージを ManabiOps に届ける仕組み。裏側の配線 |
| **クラウド** | インターネット上にあるコンピュータ。自分でサーバーを買わなくていい |
| **Google Sheets** | Google が提供する Excel のようなもの。無料で使える |
| **Firestore (ファイアストア)** | データを安全に保管する Google のデータベース |
| **Document AI** | 紙の書類を読み取って文字にしてくれる AI (OCR) |
| **Gemini (ジェミニ)** | Google が作った対話型AI。画像の中身も理解できる |
| **Instagram Graph API** | Instagram に外部から投稿するための公式の仕組み |
| **顔ぼかし** | 写真の中の顔を自動でモザイク・ぼかし加工すること |
| **暗号化** | 他人に読めないように文字を変換すること。鍵がないと戻せない |
