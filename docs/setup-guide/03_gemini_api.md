# 03. Gemini API キー取得

> 所要時間: **10分**
> 費用: **¥0** (月 45万リクエスト程度まで無料)
> このガイドで取得するもの: `GEMINI_API_KEY`

---

## 📝 このガイドでやること

1. Google AI Studio にアクセス
2. API キーを作成
3. メモ帳に記録

### ゴール
✅ AI (Gemini) を呼び出すための合言葉が手に入る

---

## 🚶 手順

### Step 1. Google AI Studio にアクセス

1. ブラウザで以下を開く
   👉 **https://aistudio.google.com/app/apikey**
2. **ガイド02 と同じ Gmail アカウント** でログイン
3. 利用規約に同意 (初回のみ)

---

### Step 2. API キーを作成

1. 画面左上または中央に **「Create API key」** (または「APIキーを作成」) ボタン → クリック
2. ポップアップで **「Create API key in existing project」** を選択
3. プロジェクトのプルダウンで **`manabi-ops`** (ガイド02で作成) を選択
4. **「Create API key」** をクリック
5. 作成されたキー (例: `AIzaSyD...` で始まる 40文字くらい) が表示される → **コピー** する

メモ帳に追記:
```
=== Gemini API ===
GEMINI_API_KEY = AIzaSyD... (ここに貼り付け)
```

---

### Step 3. 使用するモデルを確認

ManabiOps は **Gemini 2.5 Flash** を使用します (要件定義書第4章 FR-01)。

- 画面左メニューの **「Models」** または **「モデル」** でモデル一覧を確認できます
- `gemini-2.5-flash` が利用可能であることを確認
- このモデルは高速・低コスト・マルチモーダル対応 (画像理解OK)

---

### Step 4. 動作確認 (任意・推奨)

1. AI Studio のトップで **「Create new prompt」** または **「Chat」** をクリック
2. 右上のモデルを `gemini-2.5-flash` に設定
3. 日本語でテストメッセージを入力:
   ```
   こんにちは。レシートの画像から勘定科目を推論するタスクを任されています。
   ```
4. AIが返答すれば API は正常に動作しています

## 💡 豆知識

### Gemini API の無料枠
- **無料枠**: 15 RPM (1分あたり15リクエスト) / 100万トークン/日 / 1,500リクエスト/日
- **こども食堂の規模**: 月4回 × 画像30枚程度なら完全に無料枠内

### API キーの管理ベストプラクティス
- **本番用キー**と**開発用キー**は分けるのが理想 (今回は1つでOK)
- 万一漏洩したら → AI Studio の API key 画面で「Delete」してすぐ作り直し

### 料金監視
- ガイド02で設定した月額予算アラート (¥10,000) がここでも効きます
- 想定外の使用があっても自動で通知が来るので安心

## 🆘 よくあるトラブル

### Q1. 「Create API key」ボタンが見つからない
- ログインしている Google アカウントを確認
- 一旦ログアウトして、ガイド02と同じアカウントでログインし直す

### Q2. プロジェクト選択で `manabi-ops` が出てこない
- ガイド02のStep3 (プロジェクト作成) が完了しているか確認
- ブラウザをリロード (5分ほど反映に時間がかかる場合あり)

### Q3. 「API key creation failed」エラー
- ガイド02のStep6で **「Generative Language API」** を有効化したか確認
- していなければ GCP Console で有効化してから再試行

### Q4. APIキーが "Expired" になった
- 無料枠の API キーは期限なし
- もし期限付きで作成してしまった場合は削除して作り直し
