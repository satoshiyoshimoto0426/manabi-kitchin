## ✅ 事前準備チェックリスト

作業を始める前に以下を準備してください。

### 必須

- [ ] **パソコン** (Windows / Mac どちらでもOK)
- [ ] **インターネット環境** (作業中は Zoom 等の重いアプリは閉じてください)
- [ ] **Gmail アカウント** 1つ
  - 運営団体の共用アカウントを推奨 (個人アカウントだと退任時に面倒)
  - まだ無い場合は [https://accounts.google.com/signup](https://accounts.google.com/signup) で作成 (10分)
- [ ] **クレジットカード** 1枚
  - GCP の請求用。**無料枠内なら引き落としは発生しません**が、登録は必須
  - 個人カードでも可 (団体カードがベター)
- [ ] **スマホ** (LINE が入っているもの)
  - LINE公式アカウントの管理・動作確認に使用
- [ ] **本ガイドを表示する端末** (パソコン以外、例: スマホ・タブレット)
  - 手順を見ながら作業するため、画面がもう1つあると超便利

### あるとベター

- [ ] **メモ帳アプリ** (パスワード保存用)
  - 取得する API キーをコピーしておくため
  - **セキュリティ重要**: 1Password, Bitwarden など**パスワードマネージャー**を推奨
  - 最低限: PCのメモ帳に一時保存 → 作業完了後に削除
- [ ] **クラウドストレージ** (Google Drive など)
  - サービスアカウントJSONファイルの保管用

## 🚨 セキュリティに関する重要な注意

本ガイドで取得する **API キー / サービスアカウント JSON** は、子どもたちの個人情報にアクセスできる**非常に重要な情報**です。以下を必ず守ってください。

### やっていいこと
- ✅ 1Password などのパスワードマネージャーに保存
- ✅ 運営代表者と IT 担当者の 2名以内で共有
- ✅ Secret Manager (ガイド 08) に登録して本番環境で使う

### 絶対にやってはいけないこと
- ❌ **メール / LINE / SNS に貼り付けて送る** (絶対NG)
- ❌ **GitHub などの公開リポジトリにコミット** (絶対NG)
- ❌ **付箋に書いて PC に貼る** (情弱扱いされます)
- ❌ **不要になったキーを放置** (必ず無効化する)

## 💰 費用の考え方

### 初期費用
- **¥0** (クラウドサービスは使った分だけ課金)

### 月額費用 (想定: 月4回開催・画像50枚)
- **¥500〜¥3,000** (要件定義書の目標 ¥10,000 以内に収まる想定)

### 無料枠の範囲
各サービスには **無料枠** があります:
- Gemini API: 月 45万リクエスト程度は無料
- Document AI: 月1,000ページまで無料
- Cloud Run: 月200万リクエストまで無料
- Firestore: 月2万書込 × 30日 = 月60万書込まで無料

**こども食堂の運用規模 (月4回 × 画像30枚程度) では、ほぼ無料枠内に収まります**。

### 予算上限を設定する (推奨)
GCPで月額予算アラートを設定できます。ガイド02で実施します。
**¥5,000 で警告、¥10,000 で機能停止** などに設定しておくと安心です。

## 📝 進め方の推奨

### パターンA: 一気に1日で (集中型)
朝10時〜夕方16時 × 6時間確保して、Phase 1 まで一気に構築。
→ 推奨: 運営代表者 + IT 担当者で集まって作業

### パターンB: 1日1ガイドずつ (じっくり型)
毎日夜に1時間ずつ、14日間で完成。
→ 推奨: 1人で作業・平日夜しか時間が取れない場合

### パターンC: Phase分割 (慎重型)
まず Phase 1 (経理) だけ構築 → 1ヶ月運用 → Phase 2 (名簿) → 1ヶ月運用 → Phase 3 (SNS)
→ 推奨: リスクを最小化したい場合、要件定義書の推奨
