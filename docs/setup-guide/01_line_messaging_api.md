# 01. LINE Messaging API セットアップ

> 所要時間: **30分**
> 費用: **¥0** (月200通までの無料プラン)
> このガイドで取得するもの: `LINE_CHANNEL_SECRET` と `LINE_CHANNEL_ACCESS_TOKEN`

---

## 📝 このガイドでやること

1. LINE Business ID を作る
2. プロバイダー (運営団体の箱) を作る
3. Messaging API チャネル (公式アカウント本体) を作る
4. **チャネルシークレット** と **チャネルアクセストークン** を取得する
5. メモ帳に2つの値を記録する

### ゴール

✅ 「まなびキッチン」という LINE 公式アカウントが作られて
✅ プログラムから操作できる **2つの合言葉 (シークレット/トークン)** が手に入る

---

## 🚶 手順

### Step 1. LINE Developers コンソールを開く

1. パソコンのブラウザで以下のURLを開く
   👉 **https://developers.line.biz/ja/**
2. 画面右上の **「ログイン」** をクリック
3. 「ビジネスアカウントでログイン」を選択
   - まだアカウントが無い場合 → **「LINE Business ID を作成」** をクリック
   - 画面の指示に従って作成 (メール or LINEアカウントで作成可)

---

### Step 2. プロバイダーを作成する

**プロバイダーとは**: 公式アカウントを管理する「運営団体の箱」。1つ作れば複数のLINE公式アカウントを入れられます。

1. ログイン後、トップページに **「プロバイダー」** というセクションが表示される
2. **「作成」** ボタンをクリック (初回は「新規プロバイダー作成」)
3. プロバイダー名に **「まなびキッチン」** と入力
4. **「作成」** をクリック

> 📌 プロバイダー名は後から変更できます。迷ったら団体名でOK。

---

### Step 3. Messaging API チャネルを作成する

**チャネルとは**: LINE公式アカウントの本体。これが「友だち追加」できる対象になります。

1. プロバイダー画面に入ったら **「チャネルを作成」** の一覧から **「Messaging API」** を選択
2. 以下の項目を入力:

   | 項目 | 入力する値 |
   |------|-----------|
   | **チャネルの種類** | Messaging API (選択済のはず) |
   | **プロバイダー** | まなびキッチン (選択済のはず) |
   | **会社・事業者の所在国・地域** | 日本 |
   | **チャネルアイコン** | まなびキッチンのロゴ画像をアップロード (後で変更可) |
   | **チャネル名** | **まなびキッチン** |
   | **チャネル説明** | 「こども食堂の運営支援システムです」など |
   | **大業種** | 「非営利団体」 or 「教育」 |
   | **小業種** | 「NPO・公益法人」など近いもの |
   | **メールアドレス** | 運営団体の連絡用メールアドレス |
   | **プライバシーポリシーURL** | 空欄でOK (後で追加可) |
   | **サービス利用規約URL** | 空欄でOK |

3. 利用規約に同意するチェックボックスをすべて ✅
4. **「作成」** をクリック

> ⏰ この時点で公開前のLINE公式アカウントが作られました。まだ友だち追加はできません。

---

### Step 4. チャネル基本情報から **チャネルシークレット** を取得

1. 作成されたチャネルの画面が開く
2. 上部のタブから **「チャネル基本設定」** を選択
3. 下にスクロールして **「チャネルシークレット」** の項目を探す
4. 表示されている文字列 (例: `a1b2c3d4e5f6g7h8...`) を **コピー** する
5. メモ帳に以下のように貼り付け:

   ```
   === ManabiOps API キー ===
   LINE_CHANNEL_SECRET = (ここに貼り付け)
   ```

> 📌 **チャネルシークレットは絶対に他人に見せない** こと。

---

### Step 5. **チャネルアクセストークン** を発行

1. 同じチャネル画面の上部タブ **「Messaging API設定」** を選択
2. 画面を一番下までスクロール
3. **「チャネルアクセストークン (長期)」** のセクションに **「発行」** ボタンがある → クリック
4. 発行された長い文字列 (200文字くらい) が表示される → **コピー**
5. メモ帳に追記:

   ```
   === ManabiOps API キー ===
   LINE_CHANNEL_SECRET = ...
   LINE_CHANNEL_ACCESS_TOKEN = (ここに貼り付け)
   ```

> 📌 このトークンは**1回しか表示されない**ので、必ずコピーしてください。
> 万一失くしても「再発行」できますが、既存の設定は全て無効になります。

---

### Step 6. 応答設定を確認する

**ここが重要**: デフォルトでは「自動応答メッセージ」が有効になっており、ManabiOps の動作を妨害します。**必ずOFFにしてください**。

1. **「Messaging API設定」** タブの中段にある **「応答設定」** を見つける
2. **「応答メッセージ」** → **無効** に変更
3. **「あいさつメッセージ」** → 有効のままでOK (初回友だち追加時の挨拶)
4. **「Webhook」** → **有効** にする
   - URL はまだ空欄でOK (ガイド 10 で設定します)

### 完了時の応答設定

| 項目 | 設定値 |
|------|:------:|
| 応答メッセージ | ❌ 無効 |
| あいさつメッセージ | ✅ 有効 (任意) |
| Webhook | ✅ 有効 |

---

### Step 7. あいさつメッセージを設定 (任意)

1. 「あいさつメッセージ」の右側の **「編集」** をクリック
2. 新規メッセージを作成。例:
   ```
   🍙 まなびキッチンの運営サポートBotへようこそ!

   レシート・名簿・イベント写真をこのトークに送信すると、
   AIが自動で整理してお知らせします。

   まずは運営代表者に、利用者登録を依頼してください。
   ```
3. 保存

---

### Step 8. QRコードと ベーシックID の取得

1. **「Messaging API設定」** タブの上部に以下の情報がある:
   - **Bot basic ID**: `@xxxxxxxxx` (友だち追加用ID)
   - **QRコード**: 画像をダウンロード

メモ帳に追記:
```
=== ManabiOps LINE 公式アカウント情報 ===
Bot basic ID: @xxxxxxxxx
QRコード: (画像ファイルを団体共有フォルダに保存)
```

これらはスタッフに「友だち追加してね」と案内する時に使います。

## 🆘 よくあるトラブル

### Q1. 「ビジネスアカウントが必要です」と表示される
→ 上記 Step 1 で LINE Business ID を作成してください。個人の LINE アカウントとは別物です。

### Q2. 「チャネルアクセストークン」の「発行」ボタンが見つからない
→ タブが **「Messaging API設定」** になっているか確認 (「チャネル基本設定」と間違えやすい)。
→ 画面を一番下までスクロールしてください。

### Q3. シークレットやトークンを失くしてしまった
- **シークレット**: 「チャネル基本設定」タブで確認可能 (失くしても表示は消えない)
- **トークン**: 「再発行」ボタンで新しいものを発行可能。**ただし古いトークンは無効になる**ので、後続ガイドで設定する前なら問題なし。

### Q4. 「料金が発生しそうで怖い」
→ フリープラン (月200通) で今は十分です。月4回開催×メッセージ50通 = 月200通程度。
→ 利用通数は LINE Official Account Manager で確認できます。
