# 07. Google Sheets 会計台帳作成

> 所要時間: **15分**
> 費用: **¥0** (Google 無料サービス)
> このガイドで取得するもの: `SHEETS_SPREADSHEET_ID`

---

## 📝 このガイドでやること

ManabiOps は、LINE で承認されたレシート取引を **Google Sheets の会計台帳** に自動追記します。
これは要件定義書 **FR-03** (会計台帳自動記帳) を実現するためです。

### ゴール

✅ 「まなびキッチン 会計台帳」スプレッドシートが作成される
✅ 1行目にヘッダー (日付・種別・勘定科目・金額・...) が入力される
✅ サービスアカウントが「編集者」として共有される
✅ `SHEETS_SPREADSHEET_ID` をメモする

---

## 🎯 前提: なぜ Firestore ではなく Sheets か?

要件定義書 **第2章** より:
> 「運営代表者が PC が苦手で、データは **普段使い慣れた Excel/Google Sheets** で見たい」

そのため、会計データは **Firestore + Google Sheets の両方** に保存する設計になっています。
- **Firestore**: システムの内部データベース (高速・集計用)
- **Google Sheets**: 人間が見る・CSV出力する用

---

## Step 1. 新しいスプレッドシートを作成

1. [Google Sheets](https://sheets.google.com/) を開く
   - **運営代表者の Google アカウント** でログイン (他の人のアカウントでは❌)
2. 左上の **「+ 空白」** (Blank) をクリック

### 1-1. ファイル名を変更

- 画面左上の **「無題のスプレッドシート」** をクリック
- 名前を **`まなびキッチン 会計台帳`** に変更 (日本語OK)
- Enterキーで確定

---

## Step 2. シート名を変更

- 画面下部のタブ **「シート1」** をダブルクリック
- **`ledger`** に変更 (⭐ 半角英字で `ledger`)
- Enterキーで確定

> 💡 `ledger` という名前は ManabiOps コード内で固定されているため、**必ず半角英字で `ledger`** にしてください。

---

## Step 3. ヘッダー行を入力

1行目に以下の **10列** を入力します。

| A | B | C | D | E | F | G | H | I | J |
|---|---|---|---|---|---|---|---|---|---|
| 日付 | 種別 | 勘定科目 | 金額 | 店舗 | 品目 | 取引ID | 承認者 | 承認日時 | レシートURL |

### 入力手順

1. A1 セルをクリック
2. 以下の1行を全コピー → A1 に貼り付け (タブ区切りで一度に入る)

```
日付	種別	勘定科目	金額	店舗	品目	取引ID	承認者	承認日時	レシートURL
```

3. ヘッダー行を太字に: A1:J1 を選択 → **Ctrl+B** (Macは ⌘+B)
4. ヘッダー行を固定: **「表示」** メニュー → **「固定」** → **「1行」**

---

## Step 4. スプレッドシートIDを取得

1. 画面上部のURL欄を見る:
```
https://docs.google.com/spreadsheets/d/1AbCdEfGhIjKlMnOpQrStUvWxYz0123456789/edit
                                      ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                      これが SPREADSHEET_ID
```

2. URL の `/d/` と `/edit` の間の長い文字列 (約44文字) をコピー
3. `SHEETS_SPREADSHEET_ID` としてメモしておく (`.env` で使用)

### 📝 メモ例

```
SHEETS_SPREADSHEET_ID=1AbCdEfGhIjKlMnOpQrStUvWxYz0123456789
```

---

## Step 5. サービスアカウントと共有

**[02_gcp_project.md](./02_gcp_project.md) Step 5** で作成したサービスアカウントメールアドレスに、
このスプレッドシートへの編集権限を付与します。

### 5-1. サービスアカウントのメールアドレスを確認

- 形式: `manabi-ops-sa@manabi-ops.iam.gserviceaccount.com`
- GCP Console → **IAM と管理** → **サービス アカウント** で確認できる
- **このメールアドレスをコピー**

### 5-2. スプレッドシートで共有

1. スプレッドシート右上の **「共有」** ボタン (青色) をクリック
2. 「ユーザーやグループを追加」欄に、**サービスアカウントのメールアドレス** を貼り付け
3. 右側の権限が **「編集者」** (Editor) であることを確認
4. **「通知を送信」のチェックを外す** (⭐ サービスアカウントにメールは届かないため)
5. 「ロールを確認」の警告が出たら **「通知なしで共有」** (Share anyway) をクリック
6. **「完了」** をクリック

---

## Step 6. (任意) 書式を見やすく整える

運営代表者が見やすいように、お好みで以下を設定:

- **金額列 (D列)** を「通貨」形式に:
  - D列ヘッダー (D) を右クリック → **「列全体を選択」** → **「表示形式」** → **「数値」** → **「通貨」** → **¥**
- **日付列 (A列)** を「2026/04/16」形式に:
  - A列ヘッダー → **「表示形式」** → **「数値」** → **「日付」**
- **ヘッダー行の背景色**:
  - A1:J1 選択 → 塗りつぶしアイコン → 薄い黄色など
- **行の縞模様**:
  - A1:J1000 選択 → **「表示形式」** → **「交互の背景色」** → **OK**

---

## Step 7. 集計シート (任意・便利)

月次集計を手軽に見たい場合、もう1つシートを追加します。

1. 画面下部の **「+」** アイコンで新しいシートを追加
2. シート名を **`summary`** に変更
3. 以下のセルに関数を入力:

| セル | 関数 | 意味 |
|------|------|------|
| A1 | `月` | ヘッダー |
| B1 | `収入合計` | ヘッダー |
| C1 | `支出合計` | ヘッダー |
| D1 | `残高` | ヘッダー |
| A2 | `=TEXT(TODAY(),"yyyy/MM")` | 今月 |
| B2 | `=SUMIFS(ledger!D:D, ledger!B:B, "収入", ledger!A:A, ">="&EOMONTH(TODAY(),-1)+1)` | 今月の収入 |
| C2 | `=SUMIFS(ledger!D:D, ledger!B:B, "支出", ledger!A:A, ">="&EOMONTH(TODAY(),-1)+1)` | 今月の支出 |
| D2 | `=B2-C2` | 今月の残高 |

→ ManabiOps がデータを追記する度に自動で集計されます。

## 🆘 トラブルシューティング

### ❌ サービスアカウントへの共有でエラー

- メールアドレスのコピー間違いがないか確認 (`@manabi-ops.iam.gserviceaccount.com` で終わる形式)
- 「通知を送信」のチェックを外すことを忘れずに

### ❌ スプレッドシートIDが分からない

- URL全体を運営代表者・開発担当者に送れば読み取れます
- 形式: `https://docs.google.com/spreadsheets/d/【ここがID】/edit`

### ❌ ManabiOps デプロイ後、データが書き込まれない

- サービスアカウントの共有権限が「編集者」になっているか
- Google Sheets API が有効か ([02_gcp_project.md](./02_gcp_project.md) Step 4 参照)
- デプロイ後の Cloud Run ログを [13_troubleshooting.md](./13_troubleshooting.md) で確認
